## protein level
### HBA1 vs HBA2 
On protein level, there is no difference between HBA1 and HBA2. On ncbi both genes have a joint sequence listed. That is because on DNA level, they are different, but on protein level they don't differ! That is possible because the genetic code is redunddant, with 4 bases there are 4^3 (=64) possible codons which is way more than the aminoacids that have to be encoded. So there are multiple codons which can code for a single aminoacid. Meaning DNA can change, without necessarily changing a protein sequence.
